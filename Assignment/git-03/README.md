# Welcome to readme-md-generator👋

> ### CLI that generates beautiful README.md files.

### 🏠 Homepage
## Prerequisites
<!--OL-->
* npm >=5.50
* node >=9.30

## 🛠️ Install
 
```bash
npm install 
```
    
## 🔍 Usage

```bash
npm run start
```
### 🔬 Run test

```bash
npm run test:
```
